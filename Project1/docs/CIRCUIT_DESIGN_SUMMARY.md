# 回路設計の議論のまとめ

## 1. 目的

ミルククラウン撮影のための多機能LEDシステムを設計する。

- **トリガー用LED**: iPhoneカメラが検知するための赤色LED
- **照明用LED**: 240fps高速撮影用の高輝度白色LED
- **多目的LED**: 状態表示や演出用のフルカラーLED

## 2. 部品選定

| 役割 | 部品 | 型番 | 数量 | 評価 |
|:---|:---|:---|:---:|:---:|
| マイコン | Seeed XIAO | SAMD21 | 1 | ✅ 小型・安価 |
| トリガー用LED | 赤色LED | SML-E12V8WT86 | 6 | ✅ 適切 |
| 照明用LED | 白色LED | NSSW157T | 6 | ✅ 最適（高演色性） |
| 多目的LED | NeoPixel | WS2813B | 6 | ✅ 適切（要レベルシフト） |
| LEDドライバ | 定電流ドライバ | TLC5916IPWR | 2 | ✅ 最適（SPI制御） |
| レベルシフタ | バスバッファ | 74AHC1G125W5 | 1 | ✅ 最適（秋月で入手可） |

## 3. 回路設計のポイント

### 3.1. LEDドライバ（TLC5916IPWR）

- **制御方式**: I2Cではなく**SPI制御**。XIAOのSPIインターフェースを使用する。
- **カスケード接続**: 赤色用と白色用の2つのTLC5916をカスケード接続（SDO→SDI）することで、少ないピン数で制御可能。
- **定電流制御**: 外部抵抗（R-EXT）で各LEDに流れる電流を一定に保つ。これにより、LEDのVf（順電圧）の違いを吸収できる。

### 3.2. NeoPixel（WS2813B）のレベルシフト

- **問題**: WS2813Bは5V駆動、XIAOの信号は3.3V。電圧レベルが異なり不安定。
- **解決策**: **74AHC1G125W5**をレベルシフタとして使用する。
  - 74AHC1G125W5のVCCに5Vを供給。
  - XIAOの3.3V信号を入力すると、5V信号に変換されて出力される。
  - これにより、WS2813Bを安定して駆動できる。

### 3.3. 電源

- **計算**: 全てのLEDを最大輝度で点灯させると、約600mAの電流が必要。
- **推奨**: PCのUSBポート（500mA）では不足する可能性があるため、**5V 1A以上のACアダプタ**を使用する。

## 4. 最終的な回路構成案

```
+5V ACアダプタ
      │
      ├── XIAO SAMD21
      │     │
      │     ├── SPI → TLC5916 #1 (赤) → 赤色LED x6
      │     │
      │     ├── SPI → TLC5916 #2 (白) → 白色LED x6
      │     │
      │     └── D2 → 74AHC1G125W5 → WS2813B x6
      │
      ├── 74AHC1G125W5 (VCC=5V)
      │
      └── WS2813B x6 (VCC=5V)
```

## 5. 段階的な実装案

1. **ステップ1: NeoPixelのみで基本を学ぶ**
   - XIAO + WS2813B + 74AHC1G125W5 の最小構成で、LED制御の基本を習得する。
2. **ステップ2: 個別LEDを追加する**
   - TLC5916を追加し、赤色・白色LEDを制御する。
3. **ステップ3: 全ての機能を統合する**
   - 全てのLEDを円形基板に配置し、最終的なシステムを完成させる。
