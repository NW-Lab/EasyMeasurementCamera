# Project 3: 気流衝突による水滴変形 - システム統合回路

## 概要

Project1（LED照明）、Project2（電磁弁）、およびマイクロブロア駆動回路を統合した、Project3の完全なシステム回路です。

- **水滴生成**: 電磁弁 (D1)
- **気流生成**: マイクロブロア (D0)
- **照明**: TLC5916 LEDドライバ × 2 (D2-D9)

## 回路図 (ASCII)

```
(XIAO SAMD21)
+---------------------------------------------------------------------------------+
|                                                                                 |
| D0 ---[PWM 25kHz]---> TC4428 (Gate Driver) --[H-Bridge]--> [Micro Blower]        |
|                                                                                 |
| D1 -----------------> MOSFET Gate ------------> [Solenoid Valve] (5V)            |
|                                                                                 |
| D2 --[SDI]--+                                                                   |
| D3 --[CLK]--|--> [TLC5916 #1 - Red LED]                                         |
| D4 --[LE]---+                                                                   |
| D5 --[~OE]--'                                                                   |
|                                                                                 |
| D6 --[SDI]--+                                                                   |
| D7 --[CLK]--|--> [TLC5916 #2 - White LED]                                       |
| D8 --[LE]---+                                                                   |
| D9 --[~OE]--'                                                                   |
|                                                                                 |
| 5V -----(XIAO 5V Pin)---------------------------------+                         |
|                                                       |                         |
| GND --------------------------------------------------+--[GND]                   |
|                                                                                 |
+---------------------------------------------------------------------------------+

              +12V (for Blower Driver)
                |
+12V --+--------| VDD(8)         OUT_A(7) ----+       
       |        | TC4428         OUT_B(5) ----|-------+       
       |        |                IN_A(2) <----|---- D0
+ C1   |        |                IN_B(4) <----'       
 0.1uF |        |                GND(1,3,6) --- GND
       |        '-----------------------------------
       |
      GND
```

## ピン割り当て

| XIAOピン | 機能 | 接続先 |
|:---|:---|:---|
| **D0** | ブロアPWM | TC4428 ゲートドライバ IN_A, IN_B |
| **D1** | 電磁弁制御 | MOSFET ゲート |
| **D2** | LED(赤) SDI | TLC5916 #1 SDI |
| **D3** | LED(赤) CLK | TLC5916 #1 CLK |
| **D4** | LED(赤) LE | TLC5916 #1 LE |
| **D5** | LED(赤) ~OE | TLC5916 #1 ~OE |
| **D6** | LED(白) SDI | TLC5916 #2 SDI |
| **D7** | LED(白) CLK | TLC5916 #2 CLK |
| **D8** | LED(白) LE | TLC5916 #2 LE |
| **D9** | LED(白) ~OE | TLC5916 #2 ~OE |
| **D10**| (未使用) | - |
| **A0** | (未使用) | - |

## 電源系統

- **5V**: XIAO本体、電磁弁、TLC5916 LEDドライバ
- **12V**: マイクロブロア駆動用のTC4428ゲートドライバ

**【注意】** 5Vと12VのGNDは必ず共通に接続してください。
