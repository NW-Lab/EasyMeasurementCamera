# マリオット瓶の作り方：完全ガイド

## マリオット瓶とは

マリオット瓶（Mariotte Bottle）は、17世紀のフランスの物理学者エドム・マリオットが発明した装置で、**容器内の水位が変わっても、出口の水圧を一定に保つ**仕組みです。

水滴撮影において、この一定の水圧が非常に重要です。水圧が一定であれば、水滴のサイズと落下間隔が安定し、長時間にわたって同じ条件で撮影できます。

## 原理

マリオット瓶の原理は、以下の物理法則に基づいています。

### 水圧の決定要因

水圧は、**水面から出口までの高さ（水頭）**で決まります。

```
水圧 = ρ × g × h

ρ: 水の密度（約1000 kg/m³）
g: 重力加速度（約9.8 m/s²）
h: 水頭（水面から出口までの高さ）
```

### マリオット瓶の仕組み

通常の容器では、水が減ると水面が下がり、水頭（h）が小さくなるため、水圧が下がります。

しかし、マリオット瓶では、**エアチューブの先端が「仮想的な水面」として機能**します。水面がエアチューブの先端より下がると、空気が入って水面を維持するため、水頭が常に一定になります。

```
通常の容器                マリオット瓶
┌─────────┐              ┌─────────┐
│         │              │  ↓空気  │
│  空気   │              │    ↑    │ ← エアチューブ
│         │              │  水 ↑   │    先端（仮想水面）
├─────────┤ ← 水面      │    ↑    │
│         │              │    ↑    │
│  水     │              │    ↓    │
│    ↓    │              │    ↓    │
└────┬────┘              └────┬────┘
     ↓                        ↓
  水圧が変わる              水圧が一定
```

## 必要な材料

マリオット瓶を作るには、以下の材料が必要です。

| 材料 | 仕様 | 価格（目安） | 入手先 |
|---|---|---|---|
| ペットボトル | 500ml〜1L、透明 | 0円 | 手持ち |
| 点滴セット | 医療用輸液セット | 500円〜 | Amazon、楽天、薬局 |
| エアチューブ | 内径4〜6mm、長さ30〜50cm | 200円 | ホームセンター、熱帯魚店 |
| ホットグルー | 防水用 | 300円 | 100円ショップ、ホームセンター |
| PTFEテープ | 配管用シール材 | 200円 | ホームセンター |
| 針金または支柱 | エアチューブ固定用 | 100円 | 100円ショップ |

**合計**: 約1,300円（ペットボトル除く）

## 作り方

### ステップ1: ペットボトルの準備

透明なペットボトル（500ml〜1L）を用意します。中身を空にして、よく洗って乾かします。

**ポイント**:
- 透明なボトルを選ぶと、水位やエアチューブの位置が見えて調整しやすい
- 硬めのボトルの方が、圧力変化に強い

### ステップ2: 蓋に穴を開ける

ペットボトルの蓋に、以下の3つの穴を開けます。

1.  **点滴セット用の穴（大）**: 直径6〜8mm
2.  **エアチューブ用の穴（中）**: 直径4〜6mm
3.  **大気圧導入用の穴（小）**: 直径1〜2mm

```
蓋（上から見た図）

    ┌───────────┐
    │           │
    │  ● 小穴  │ ← 大気圧導入用（1〜2mm）
    │           │
    │  ◎ 大穴  │ ← 点滴セット用（6〜8mm）
    │           │
    │  ○ 中穴  │ ← エアチューブ用（4〜6mm）
    │           │
    └───────────┘
```

**穴の開け方**:
- 小さな穴: キリや太めの針で開ける
- 中〜大きな穴: ドリルやハンダゴテで開ける

**注意**:
- 穴を開けるときは、蓋の下に木の板などを敷いて、安全に作業する
- 穴の周りがバリバリにならないように、丁寧に仕上げる

### ステップ3: 点滴セットの取り付け

点滴セットをペットボトルの底まで届く長さに調整し、蓋の大きな穴に通します。

```
蓋（横から見た図）

    ┌───────────┐
    │           │
    └─────┬─────┘
          │
      点滴セット
          │
          ↓
    （ボトル内部）
          │
          │
          │
          ↓
       ボトルの底
```

**取り付け方**:
1. 点滴セットのチューブを蓋の穴に通す
2. PTFEテープを巻いて、隙間を埋める
3. ホットグルーで固定して、完全に密閉する

**ポイント**:
- 点滴セットの先端は、ボトルの底から1〜2cm上にする
- 底にピッタリつけると、水が少なくなったときに吸えなくなる

### ステップ4: エアチューブの取り付け

エアチューブを蓋の中くらいの穴に通します。エアチューブの長さは、**ボトル内で水面より5〜10cm上に来るように調整**します。

```
エアチューブの位置（重要！）

┌─────────────┐
│  ← 空気が入る
│             │
│    空気     │
├─────────────┤ ← 水面
│      ↑      │
│    5-10cm   │ ← この距離が水圧を決める
│      ↑      │
│   ← エアチューブの先端
│    水       │
│             │
│      ↓      │ ← 点滴セット
└──────┬──────┘
```

**取り付け方**:
1. エアチューブを適切な長さに切る
2. 蓋の穴に通す
3. PTFEテープを巻いて、隙間を埋める
4. ホットグルーで固定して、完全に密閉する

**重要なポイント**:
- **エアチューブの先端の高さが、水圧を決定します**
- 先端が低い（水面に近い）→ 水圧が低い → 水滴が小さい
- 先端が高い（水面から遠い）→ 水圧が高い → 水滴が大きい

### ステップ5: 大気圧導入用の小穴

蓋の小さな穴は、そのまま開けたままにします。この穴から大気圧が導入され、マリオット瓶が正常に機能します。

**注意**:
- この穴を塞がないこと
- 穴が小さすぎると、空気の流れが悪くなる
- 穴が大きすぎると、水が漏れる可能性がある

### ステップ6: 水密性のチェック

すべての穴をホットグルーで固定したら、水密性をチェックします。

1. ペットボトルに水を入れる（蓋は開けたまま）
2. 蓋を閉める
3. ボトルを逆さまにして、蓋から水が漏れないか確認
4. 漏れる場合は、ホットグルーを追加して補修

### ステップ7: エアチューブの高さ調整

エアチューブの先端の高さを調整します。この高さが、水圧を決定します。

**調整方法**:
1. ボトルを逆さまにして、スタンドに固定
2. 水を入れる
3. 点滴セットの調整ダイヤルを開く
4. 水滴が落ち始めたら、エアチューブから「ボコボコ」と空気が入るのを確認
5. 水滴のサイズを見ながら、エアチューブの高さを微調整

**調整のコツ**:
- エアチューブを上に引っ張る → 水圧が上がる → 水滴が大きくなる
- エアチューブを下に押す → 水圧が下がる → 水滴が小さくなる

### ステップ8: エアチューブの固定

適切な高さが決まったら、エアチューブを固定します。

**固定方法**:
- 針金や支柱でエアチューブを支える
- 蓋の外側でエアチューブをクリップで固定
- ホットグルーで完全に固定（調整が終わった後）

## 使い方

### セットアップ

1. マリオット瓶をスタンドに逆さまに固定
2. 水を入れる（蓋を一時的に開けて注ぐ）
3. 蓋を閉める
4. 点滴セットの調整ダイヤルを開く
5. 水滴が落ち始める
6. エアチューブから「ボコボコ」と空気が入るのを確認

### 水滴の間隔調整

点滴セットの調整ダイヤルを回して、水滴の間隔を調整します。

- ダイヤルを開く → 水滴の間隔が短くなる（頻繁に落ちる）
- ダイヤルを閉じる → 水滴の間隔が長くなる（ゆっくり落ちる）

30秒に1滴にしたい場合は、ダイヤルをほぼ閉じて、少しだけ開きます。

### 水の補充

水が減ったら、蓋を開けて水を補充します。マリオット瓶の利点は、水を補充しても水圧が変わらないため、調整し直す必要がないことです。

## トラブルシューティング

### 水が落ちない

**原因**: 蓋が完全に密閉されていない、または大気圧導入用の穴が塞がっている

**対策**:
- すべての穴の周りをホットグルーで再度密閉
- 大気圧導入用の小穴が開いているか確認

### 水滴のサイズが不安定

**原因**: エアチューブの高さが適切でない、または水圧が低すぎる

**対策**:
- エアチューブを上に引っ張って、水圧を上げる
- 点滴セットの調整ダイヤルを少し開く

### エアチューブから空気が入らない

**原因**: エアチューブが詰まっている、または水面より下にある

**対策**:
- エアチューブを取り外して、詰まりを確認
- エアチューブの先端が水面より上にあるか確認

### 水が蓋から漏れる

**原因**: 穴の密閉が不十分

**対策**:
- ホットグルーを追加して、完全に密閉
- PTFEテープを多めに巻く

## 電磁弁との組み合わせ

マリオット瓶は、点滴セットだけでなく、電磁弁と組み合わせることもできます。

### 接続方法

```
マリオット瓶
    ↓
  チューブ
    ↓
  電磁弁
    ↓
  ノズル
    ↓
  水滴
```

マリオット瓶の出口（点滴セットの代わり）にチューブを接続し、電磁弁に繋ぎます。電磁弁を開くと、一定の水圧で水が出るため、精密な水滴制御が可能になります。

### 利点

- 水圧が一定なため、電磁弁の開放時間と水滴サイズの関係が安定
- 長時間の使用でも、同じ設定で同じ水滴を作れる
- イベントでの使用に最適

## まとめ

マリオット瓶は、水位が変わっても水圧を一定に保つ優れた装置です。約1,300円の材料費で作ることができ、水滴撮影において非常に有用です。

**重要なポイント**:
1. エアチューブの先端の高さが水圧を決定する
2. すべての穴を完全に密閉する
3. 大気圧導入用の小穴を忘れない
4. エアチューブから「ボコボコ」と空気が入れば、正常に動作している

イベントでの使用を考えると、マリオット瓶は最も推奨される水滴供給方法です。ぜひ、挑戦してみてください！
