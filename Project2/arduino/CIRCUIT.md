# 回路図：Project2 - 電磁弁式ミルククラウン撮影トリガー

**Project1のフォトカプラ（受動的検知）方式から、電磁弁（能動的生成）方式に設計を変更します。**

- **目的**: 電磁弁を精密に制御し、任意のタイミングで水滴を生成・落下させ、ミルククラウンを撮影する。
- **主要部品**: uxcell製 小型電磁弁 (DC 5V / 0.26A)

---

## システム全体構成

```
+5V ACアダプタ (2A以上推奨)
      │
      ├──────────┬──────────────────────────┐
      │              │                                  │
      ▼              │                                  ▼
Seeed XIAO SAMD21      ▼                               5V 電源ライン
（USBから給電）  電磁弁駆動回路                      │
      │              │                                  │
      ├── D0  → MOSFET Gate ─→ 電磁弁制御              │
      │                                                 │
      ├── D2  → SK6812 DIN → NeoPixel                TLC5916 #1, #2
      │                                                 │
      │ (TLC5916制御はProject1と同じ: 8ピン使用)        │
      │                                                 ▼
      └─────────────────────────────────────────┴─ GND共通
```

## 1. 電磁弁の駆動回路

電磁弁はコイル（誘導性負荷）であり、マイコンのGPIOピンから直接駆動することはできません。**約260mAの電流が必要**なため、MOSFETをスイッチとして使用します。

- **MOSFET**: ロジックレベルで駆動可能なNチャネルMOSFET（例: **IRLB8721**, 2N7000）
- **フライバックダイオード**: 電磁弁OFF時に発生する逆起電力から回路を保護（例: **1N4007**）
- **外部電源**: 電磁弁用に別途5V電源を用意（ACアダプタから分岐）

### 接続方法

```
          +5V (外部電源)
           │
           ├──────────┐
           │          │
           │          ▼ ダイオード（アノード）
         電磁弁       │
           │          ▲ ダイオード（カソード）
           │          │
           ├──────────┤
           │          │
           │        MOSFET (Drain)
           │
XIAO D0 ───[1kΩ]── (Gate)
           │
         (Source) ─── GND
           │
          GND
```

- **XIAO D0** からのHIGH信号でMOSFETがONになり、電磁弁に電流が流れて開きます。
- フライバックダイオードは、**電磁弁と並列に、かつ電源側がカソードになる向き**で接続します。

## 2. XIAO SAMD21 ピン割り当て - Project2

| XIAOピン | 接続先 | 機能 |
|:---:|:---|:---|
| **D0 (A0)** | **MOSFET Gate** | **電磁弁制御信号** |
| D1 (A1) | (未使用) | |
| D2 (A2) | SK6812 DIN | NeoPixel データ信号 |
| D3 (A3) | TLC5916 #2 SDI | 白色LED データ |
| D4 (A4) | TLC5916 #2 CLK | 白色LED クロック |
| D5 (A5) | TLC5916 #2 LE | 白色LED ラッチ |
| D6 (A6) | TLC5916 #1 SDI | 赤色LED データ |
| D7 (A7) | TLC5916 #1 CLK | 赤色LED クロック |
| D8 (A8) | TLC5916 #1 LE | 赤色LED ラッチ |
| D9 (A9) | TLC5916 #1 ~OE | 赤色LED 出力イネーブル |
| D10 (A10)| TLC5916 #2 ~OE | 白色LED 出力イネーブル |

- **注意**: 電磁弁の仕様がDC 12Vの場合は、駆動回路の電源を+12Vに変更してください。MOSFET（IRLB8721等）とダイオード（1N4007）はそのままで対応可能です。
