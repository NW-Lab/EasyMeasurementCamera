# 回路図：ミルククラウン撮影トリガーシステム

## システム全体構成

```
+5V ACアダプタ（1A以上推奨）
      │
      ├─────────────────────────────────────────┐
      │                                         │
      ▼                                         ▼
Seeed XIAO SAMD21                        5V 電源ライン
（3.3V動作）                                    │
      │                                    ┌───┴────┐
      ├── D0  ← フォトカプラ（水滴検知）   │        │
      │                                    ▼        ▼
      ├── D2  → 74AHC1G125W5 → WS2813B×6  TLC5916  TLC5916
      │         （レベルシフタ）           #1(赤)   #2(白)
      │                                    │        │
      ├── MOSI(D10) → TLC5916 #1 SDI       ▼        ▼
      │               → #1 SDO → #2 SDI  赤色LED×6 白色LED×6
      │
      ├── SCK(D8)  → TLC5916 #1/#2 CLK（共通）
      ├── D3       → TLC5916 #1 LE（赤色ラッチ）
      ├── D4       → TLC5916 #2 LE（白色ラッチ）
      ├── D5       → TLC5916 #1/#2 OE（共通、GNDでも可）
      └── GND      → 全GND共通
```

---

## 1. フォトカプラ（水滴検知センサー）

### 推奨部品
- **PC817**（秋月電子 I-00204、20円）

### 回路図

```
         3.3V
          │
         1kΩ
          │
          ├─────────────── XIAO D0（INPUT_PULLUP）
          │
PC817 Collector（4番ピン）
PC817 Emitter（3番ピン） ─── GND

PC817 Anode（1番ピン） ─── 赤外線LED（+）
PC817 Cathode（2番ピン） ─── 赤外線LED（−）を経由してGND
```

### 動作原理

| 状態 | 赤外線LED | フォトトランジスタ | XIAO D0 |
|:---|:---:|:---:|:---:|
| 水滴なし | 光が届く | ON（導通） | LOW |
| **水滴通過** | **光が遮られる** | **OFF（非導通）** | **HIGH → トリガー！** |

## 2. WS2813B（NeoPixel 6連）+ レベルシフタ

### 推奨部品
- **WS2813B 6連テープ** または 個別WS2813B × 6
- **74AHC1G125W5**（秋月電子 117605、100円/5個入り）

### ⚠️ なぜレベルシフタが必要か

WS2813BのDIN信号は、VCC（5V）の70%以上（= 3.5V以上）が必要です。  
XIAOの出力は3.3Vのため、そのまま接続すると**不安定**になります。  
74AHC1G125W5を使うことで、3.3V → 5V に変換して確実に動作させます。

### 74AHC1G125W5 ピン配置（SOT23-5）

```
      ┌───┐
 ~OE ─┤1 5├─ VCC（5V）
  A  ─┤2 4├─ Y（出力 → WS2813B DIN）
 GND ─┤3  ├
      └───┘
```

| ピン | 接続先 | 説明 |
|:---:|:---|:---|
| 1 (~OE) | GND | GNDに接続で常時イネーブル |
| 2 (A) | XIAO D2 | 3.3V信号入力 |
| 3 (GND) | GND | グランド |
| 4 (Y) | WS2813B DIN | 5V信号出力 |
| 5 (VCC) | 5V | 5V電源（出力電圧を決定） |

### 回路図

```
XIAO D2（3.3V）─── 74AHC1G125W5 A(2番)
                   74AHC1G125W5 VCC(5番) ─── 5V
                   74AHC1G125W5 ~OE(1番) ─── GND
                   74AHC1G125W5 GND(3番) ─── GND
                   74AHC1G125W5 Y(4番) ────── WS2813B DIN

5V ─────────────────────────────────────── WS2813B VCC
GND ────────────────────────────────────── WS2813B GND
```

### WS2813B LED配置（円形 6個）

```
        [LED 0]
    [LED 5]  [LED 1]
    [LED 4]  [LED 2]
        [LED 3]

LED 0-2: 赤色モード時に点灯（トリガー用）
LED 3-5: 白色モード時に点灯（照明用）
※ ソフトウェアで自由に変更可能
```

## 3. TLC5916IPWR（LEDドライバ）

### 推奨部品
- **TLC5916IPWR**（Mouser、Digikey、約100円）
- パッケージ: TSSOP-16（表面実装）

### TLC5916IPWR ピン配置（TSSOP-16）

```
        ┌────────┐
  SDI  ─┤1    16├─ VDD（3.3V または 5V）
  CLK  ─┤2    15├─ R-EXT（電流設定抵抗）
  LE   ─┤3    14├─ VLED（LED電源、5V）
  ~OE  ─┤4    13├─ OUT7
  OUT0 ─┤5    12├─ OUT6
  OUT1 ─┤6    11├─ OUT5
  OUT2 ─┤7    10├─ OUT4
  GND  ─┤8     9├─ OUT3
        └────────┘
```

### ピン説明

| ピン | 名称 | 説明 |
|:---:|:---|:---|
| 1 | SDI | シリアルデータ入力（XIAO MOSIから） |
| 2 | CLK | クロック入力（XIAO SCKから） |
| 3 | LE | ラッチイネーブル（HIGHで出力更新） |
| 4 | ~OE | 出力イネーブル（LOWで出力ON） |
| 5-13 | OUT0-7 | LED出力（定電流シンク） |
| 15 | R-EXT | 外部抵抗で電流値を設定 |
| 14 | VLED | LED電源（5V） |
| 16 | VDD | IC電源（3.3Vまたは5V） |

### 電流設定抵抗（R-EXT）の計算

TLC5916の出力電流は以下の式で決まります：

```
I_OUT = 15 × (1.24V / R-EXT)
```

| R-EXT | 出力電流 | 用途 |
|:---:|:---:|:---|
| 1.0kΩ | 18.6mA | 標準（推奨） |
| 1.2kΩ | 15.5mA | やや暗め・省電力 |
| 750Ω | 24.8mA | 高輝度（LED定格内） |

**推奨**: 1.0kΩ（赤色・白色ともに同じ値でOK）

### カスケード接続（2個のTLC5916を接続）

```
XIAO MOSI(D10) ─── TLC5916 #1 SDI
                   TLC5916 #1 SDO ─── TLC5916 #2 SDI
                   TLC5916 #2 SDO ─── （未接続）

XIAO SCK(D8) ──── TLC5916 #1 CLK
                   TLC5916 #2 CLK（並列接続）

XIAO D3 ────────── TLC5916 #1 LE（赤色ラッチ）
XIAO D4 ────────── TLC5916 #2 LE（白色ラッチ）

XIAO D5 ────────── TLC5916 #1 ~OE
                   TLC5916 #2 ~OE（並列接続）
※ ~OEをGNDに固定すれば常時出力ON（ピン節約）
```

### 赤色LED（SML-E12V8WT86）接続

```
TLC5916 #1 OUT0 ─── 赤色LED #1 カソード（−）
TLC5916 #1 OUT1 ─── 赤色LED #2 カソード（−）
TLC5916 #1 OUT2 ─── 赤色LED #3 カソード（−）
TLC5916 #1 OUT3 ─── 赤色LED #4 カソード（−）
TLC5916 #1 OUT4 ─── 赤色LED #5 カソード（−）
TLC5916 #1 OUT5 ─── 赤色LED #6 カソード（−）

5V ─────────────── 赤色LED #1〜#6 アノード（+）共通
```

### 白色LED（NSSW157T）接続

```
TLC5916 #2 OUT0 ─── 白色LED #1 カソード（−）
TLC5916 #2 OUT1 ─── 白色LED #2 カソード（−）
TLC5916 #2 OUT2 ─── 白色LED #3 カソード（−）
TLC5916 #2 OUT3 ─── 白色LED #4 カソード（−）
TLC5916 #2 OUT4 ─── 白色LED #5 カソード（−）
TLC5916 #2 OUT5 ─── 白色LED #6 カソード（−）

5V ─────────────── 白色LED #1〜#6 アノード（+）共通
```

## 5. 電源設計

### 消費電流計算

| 部品 | 最大電流 |
|:---|:---:|
| 赤色LED × 6（20mA × 6） | 120mA |
| 白色LED × 6（20mA × 6） | 120mA |
| WS2813B × 6（全白点灯、60mA × 6） | 360mA |
| XIAO SAMD21 | 約50mA |
| **合計** | **約650mA** |

**推奨電源**: 5V 1A以上のACアダプタ（USB-Cタイプ推奨）

### 注意事項

- 全LEDを同時に最大輝度で点灯させることは稀なため、実際の消費電流は300〜400mA程度になる見込みです。
- WS2813Bの輝度をソフトウェアで50%以下に制限すると、大幅に消費電流を削減できます。

## 7. トラブルシューティング

### WS2813Bが点灯しない・不安定
- 74AHC1G125W5のVCC（5番ピン）が5Vになっているか確認
- ~OE（1番ピン）がGNDに接続されているか確認
- DINの配線が30cm以内か確認

### 赤色・白色LEDが点灯しない
- TLC5916の~OEがLOW（GND）になっているか確認
- R-EXTが正しく接続されているか確認（1.0kΩ）
- LEDの極性（アノード/カソード）を確認

### フォトカプラが反応しない
- 赤外線LEDが点灯しているか確認（スマホカメラで確認可能）
- D0がINPUT_PULLUPに設定されているか確認
- 光軸が正しく合っているか確認
