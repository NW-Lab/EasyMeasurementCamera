# 回路図：ミルククラウン撮影トリガーシステム v2.0

**ユーザーからの指摘に基づき、TLC5916の制御方法を修正しました。**

- **修正点**: 2つのTLC5916のLE（ラッチイネーブル）ピンは個別制御ではなく、**共通のピンに接続**して同時にラッチするのが正しい仕様です。これにより、赤色LEDと白色LEDの完全な同時点灯が可能になります。

## システム全体構成

```
+5V ACアダプタ（1A以上推奨）
      │
      ├─────────────────────────────────────────┐
      │                                         │
      ▼                                         ▼
Seeed XIAO SAMD21                        5V 電源ライン
（3.3V動作）                                    │
      │                                    ┌───┴────┐
      ├── D0  ← フォトカプラ（水滴検知）   │        │
      │                                    ▼        ▼
      ├── D2  → 74AHC1G125W5 → WS2813B×6  TLC5916  TLC5916
      │         （レベルシフタ）           #1(赤)   #2(白)
      │                                    │        │
      ├── MOSI(D10) → TLC5916 #1 SDI       ▼        ▼
      │               → #1 SDO → #2 SDI  赤色LED×6 白色LED×6
      │
      ├── SCK(D8)  → TLC5916 #1/#2 CLK（共通）
      ├── D3       → TLC5916 #1/#2 LE（共通ラッチ）  <-- ★修正点
      ├── D4       → TLC5916 #1/#2 ~OE（共通、GNDでも可）
      └── GND      → 全GND共通
```

## 3. TLC5916IPWR（LEDドライバ）

### カスケード接続（2個のTLC5916を接続） - ★修正版

2つのTLC5916をカスケード（デイジーチェーン）接続することで、少ないピンで多チャンネルを制御します。

```
// データラインは直列に接続
XIAO MOSI(D10) ─── TLC5916 #1 SDI
                   TLC5916 #1 SDO ─── TLC5916 #2 SDI
                   TLC5916 #2 SDO ─── （未接続）

// クロックとラッチは並列（共通）に接続
XIAO SCK(D8) ────┬── TLC5916 #1 CLK
                   └─── TLC5916 #2 CLK

XIAO D3 ─────────┬── TLC5916 #1 LE  <-- ★共通接続
                   └─── TLC5916 #2 LE

XIAO D4 ─────────┬── TLC5916 #1 ~OE
                   └─── TLC5916 #2 ~OE
```

### 制御シーケンス（16ビット転送）

1.  **データ送信**: `SPI.transfer()`を**2回**連続で呼び出します。
    1.  `SPI.transfer(白色LEDデータ)`: 最初に送った8bitデータは、チェーンの末端にある**TLC5916 #2 (白色)** に送られます。
    2.  `SPI.transfer(赤色LEDデータ)`: 次に送った8bitデータは、**TLC5916 #1 (赤色)** に格納されます。
2.  **ラッチ**: 16ビットのデータを送り終えた後、**共通のLEピン(D3)を一度だけHIGH→LOW**します。これにより、2つのICの出力が**完全に同時に**更新されます。

## 4. XIAO SAMD21 ピン割り当て一覧 - ★修正版

| XIAOピン | 接続先 | 機能 |
|:---:|:---|:---|
| D0 | フォトカプラ Collector | 水滴検知入力（INPUT_PULLUP） |
| D2 | 74AHC1G125W5 A | WS2813B データ信号 |
| D3 | **TLC5916 #1/#2 LE** | **共通ラッチイネーブル** |
| D4 | TLC5916 #1/#2 ~OE | 出力イネーブル（共通） |
| D8 (SCK) | TLC5916 #1/#2 CLK | SPI クロック（共通） |
| D10 (MOSI) | TLC5916 #1 SDI | SPI データ |
| 5V | WS2813B VCC, TLC5916 VLED, 74AHC1G125W5 VCC | 5V電源 |
| 3.3V | TLC5916 VDD | IC電源 |
| GND | 全GND | 共通グランド |
