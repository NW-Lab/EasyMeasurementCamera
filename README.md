# EasyShareCamera
EasyShareCamera

# 目的
様々な物理現象を撮影するカメラを作成する。
教育用として同じ物理現象を受講者にもカメラで撮影してもらう。
シャッターのタイミングやカメラの設定はなかなか難しいので、Masterカメラが設定した内容が受講者用のカメラにコピーされたり、シャッター等もMasterカメラの操作に従う。
カメラの機能は、写真撮影もしくは動画になる。

Masterカメラは１台で、受講者用のRemotoカメラは複数台ある

# デバイス
デバイスはiPhone,もしくはiPadする。

# 機能
シャッターとやvideoの撮影の遅延はなるべく少なくする。（最優先）
将来的にはシャッターはマイコン等で操作する可能性がある。
UIはなるべくAppleのカメラに合わせる。
AVFoundation フレームワークを使用する。
特にスロー撮影（ハイスピード撮影）が使いたい
撮影したデータは普通の写真に保存される。

画像の共有機能はこのカメラには不要です。写真アプリに任せたらいい
