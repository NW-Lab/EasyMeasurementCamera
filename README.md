# EasyMeasurementCamera

ミルククラウン撮影用の240fpsスローモーションカメラアプリと、様々な物理現象を可視化するハードウェアトリガーシステムの実験プロジェクト群です。

## プロジェクト構成

本リポジトリは、水滴や気流などの物理現象をハイスピード撮影するための複数のプロジェクト（Project 1〜6）で構成されています。

### [Project 1] フォトカプラ検知 × LED照明
**概要:** 落下する水滴をフォトカプラで検知し、赤色LEDを点灯させてスマホの録画をトリガーする基本システム。
- **ディレクトリ:** `Project1/`
- **特徴:** マリオット瓶による自然落下、XIAO SAMD21 + TLC5916によるLED制御
- **資料:** [Project1_Slides.pdf](Project1/Project1_Slides.pdf)

### [Project 2] 電磁弁による水滴生成
**概要:** 水滴の落下タイミングを電磁弁（ソレノイドバルブ）で制御し、より正確なタイミングでミルククラウンを撮影するシステム。
- **ディレクトリ:** `Project2/`
- **特徴:** NチャネルMOSFETによる電磁弁制御、シリアル通信によるトリガー
- **資料:** [Project2_Slides.pdf](Project2/Project2_Slides.pdf)

### [Project 3] 気流衝突による水滴変形
**概要:** 村田製作所のマイクロブロア（流量タイプ）を使用し、落下中の水滴に横から高速気流を当てて変形・分裂する瞬間を撮影する実験。
- **ディレクトリ:** `Project3/`
- **特徴:** 高速気流のミリ秒単位のパルス制御
- **資料:** [Project3_Slides.pdf](Project3/Project3_Slides.pdf)

### [Project 4] 気流が描く水面の幾何学
**概要:** マイクロブロアの気流を水面に当て、電圧制御によって液面に幾何学模様（サイマティクス/クラドニ図形風）を描き出す実験。
- **ディレクトリ:** `Project4/`
- **特徴:** 脈動のない安定した気流による波紋形成
- **資料:** [Project4_Slides.pdf](Project4/Project4_Slides.pdf)

### [Project 5] 見えない気流を煙で描く
**概要:** マイクロブロアから噴出される高速気流やカルマン渦を、煙（線香やスモークマシン）を使って可視化しスローモーション撮影する実験。
- **ディレクトリ:** `Project5/`
- **特徴:** バックライト照明（TLC5916）を活用した流体可視化
- **資料:** [Project5_Slides.pdf](Project5/Project5_Slides.pdf)

### [Project 6] 空気圧で水滴を精密に射出する
**概要:** マイクロブロア（圧力タイプ）を使用して密閉タンクに圧力をかけ、細管からサブミリ単位の精密な水滴を射出する高度なシステム。
- **ディレクトリ:** `Project6/`
- **特徴:** 電磁弁を超える応答速度と精密なサイズ制御
- **資料:** [Project6_Slides.pdf](Project6/Project6_Slides.pdf)

## 開発環境
- **iOSアプリ:** Swift, SwiftUI, AVFoundation (iOS 17.0+)
- **ハードウェア制御:** Seeed Studio XIAO SAMD21, Arduino IDE
- **主要コンポーネント:** TLC5916 (LEDドライバ), TCRT5000 (フォトカプラ), IRLB8721 (MOSFET), 村田製作所マイクロブロア (MZB1001T02 / MZB3004T04)
